# ERP Padaria Pro - v4.0
Sistema de gestão industrial para padarias, focado em controle de estoque e Business Intelligence (BI).

## 🚀 Funcionalidades
- **Gestão de Vendas:** Registro de pedidos com baixa automática de insumos.
- **Módulo de BI:** Relatórios de faturamento por dia, mês e ano com cálculo de Ticket Médio.
- **Persistência de Dados:** Uso de JSON para estoque e TXT para logs de auditoria.
- **Arquitetura Modular:** Código dividido em módulos (Interface, Processamento e Dados).

## 🛠️ Tecnologias
- Python 3.x
- JSON (Persistência de dados)
